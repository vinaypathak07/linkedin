## Contributors of `Hacktoberfest 2023`

## INSTRUCTIONS -

- Fork this Repository using the button at the top on right corner.
- Clone your forked repository to your local ( git clone "url from clone options".)
- Run the index.html in your local. Go thorough the code.
- Raise the issue for the fix you are doing or enhancement/feature you are implementing.
- Wait for issue to get assigned.
- Create a new branch for your modifications (ie. `git checkout -b user-feature or user-fix`)
- Add your files (`git add -A`), commit (`git commit -m "Commit Message"`) and push (`git push origin new-user`)
- Create a pull request
- Ask to review the resolved issue or modification.
- After the review PR will be merged.
